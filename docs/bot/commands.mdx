## Troubleshooting

<AccordionGroup>
  <Accordion title="Command not working">
    **Try:**
    1. Type `/start` to reset
    2. Check for typos (commands are case-sensitive)
    3. Restart Telegram app
    4. Contact support if persists
  </Accordion>

  <Accordion title="Buttons not responding">
    **Causes:**
    - Old message (buttons expire after 48h)
    - Bot restarted (sessions cleared)
    
    **Solution:** Type `/start` to get fresh menu.
  </Accordion>

  <Accordion title="Missing commands">
    **Possible reasons:**
    - Bot not fully initialized
    - Telegram cache issue
    
    **Solution:**
    1. Send any message to bot
    2. Type `/start`
    3. All commands should appear in menu
  </Accordion>
</AccordionGroup>
