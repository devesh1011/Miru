## Why DeepMirror?

### On-Chain Copy Trading

Traditional copy trading platforms are centralized and custodial. DeepMirror executes everything on-chain through DeepBook V3, giving you:

- Full transparency of every trade
- Non-custodial control of your funds
- Direct settlement on Sui blockchain

### zkLogin Integration

No seed phrases to manage. DeepMirror uses Sui's zkLogin to create wallets tied to your Google account:

- Sign in with familiar OAuth flow
- Cryptographic security without private key management
- Optional wallet export for full custody

### Built for Speed

DeepMirror monitors maker positions in real-time and executes copy trades within milliseconds:

- Sub-second position detection
- Immediate order placement on DeepBook V3
- Telegram notifications for every action

## Platform Overview

DeepMirror operates on **Sui Testnet** and integrates with:

- **DeepBook V3** - On-chain central limit order book
- **Sui zkLogin** - Google OAuth authentication
- **Supabase** - PostgreSQL database for position tracking
- **Telegram Bot API** - Interactive command interface

<Note>
  DeepMirror is currently on Sui **Testnet**. Use testnet SUI and DEEP tokens
  only.
</Note>

## Security & Custody

Your security is paramount:

- **Non-custodial** - Bot never holds your private keys
- **Capability-based** - Grant specific permissions, revoke anytime
- **Zero-knowledge proofs** - zkLogin uses ZK proofs for authentication
- **Open source** - Fully auditable codebase

<Warning>
  Always verify wallet addresses and capability objects before approving
  transactions. DeepMirror will never ask for your private key.
</Warning>
